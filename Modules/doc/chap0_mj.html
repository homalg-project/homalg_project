Modules

A homalg based Package for the Abelian Category of Finitely Presented Modules over Computable Rings

Version 2017.07.17

September 2015

(this manual is still under construction)

This manual is best viewed as an HTML document. The latest version is available online at:

http://homalg.math.rwth-aachen.de/~barakat/homalg-project/Modules/chap0.html

An offline version should be included in the documentation subfolder of the package. This package is part of the homalg-project:

http://homalg.math.rwth-aachen.de/index.php/core-packages/modules-package
